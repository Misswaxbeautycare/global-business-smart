Comment installer l'application sur mon téléphone ?
Sur Android (Chrome) : ouvrez ce site, puis appuyez sur le bouton "Installer l'application" qui clignote en bas de l'écran. S'il n'apparaît pas, ouvrez le menu ⋮ de Chrome puis "Installer l'application".

Sur iPhone (Safari) : ouvrez ce site dans Safari, appuyez sur le bouton Partager (carré avec flèche vers le haut), puis choisissez "Sur l'écran d'accueil".

Conseil : créez d'abord votre compte dans le navigateur, installez l'application ensuite — vos données restent les mêmes, l'installation ne fait qu'ajouter une icône pratique sur votre téléphone.
